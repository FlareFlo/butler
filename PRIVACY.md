# Butler Privacy Policy

*Last updated: June 11, 2026*

## 1. What data we collect

When you add Butler to your Discord server, we store:

- **Server ID** — to associate configuration settings (logging channel, minimum account age, honeypot channels, safe roles)
- **User IDs** — logged in the moderation journal when moderation actions are taken (auto-kick, ban)
- **Role and Channel IDs** — for honeypot whitelist and logging configuration
- **Moderation action metadata** — action type, timestamp, and moderator ID (if applicable)

## 2. How we use your data

All stored data is used exclusively for the bot's moderation features:

- Server configuration to apply per-server settings
- Moderation journal for audit trail of moderation actions
- No data is shared with third parties
- No data is used for training AI/ML models

## 3. Data retention

- Data is stored only while Butler remains a member of your server
- When Butler is removed from a server, all associated data is automatically deleted
- We do not retain data after it is no longer needed for its stated purpose

## 4. Data deletion requests

You can request deletion of your data by:

- Contacting the bot operator via Discord
- Removing Butler from your server (triggers automatic cleanup)

## 5. Data security

- All data is stored on encrypted ZFS volumes (encrypted at rest)
- Access is limited to authorized bot operators
- We follow industry-standard security practices

## 6. Contact

For questions or deletion requests, reach out via Discord.

## 7. Changes

We may update this policy as needed. Continued use of Butler after changes constitutes acceptance.
